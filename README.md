## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Ejecutar en desarrollo

```bash
# clonar repositorio
- ejecutar
$ yarn install

# tener nest cli instalado
$ npm i -g @nestjs/cli

# levantar la base de datos
$ docker-compose up -d
```

## Stack utilizado

```bash
# mongo db
# nest
```
